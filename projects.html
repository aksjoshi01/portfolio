15-418/15-618: Parallel Computer Architecture and Programming

    Parallel Wire Routing with OpenMP
    • Parallelized a wire routing algorithm for optimal wire placements and minimal overlaps.
    • Parallelization at two levels: within wires for faster route generation and across wires for concurrent processing.
    • Achieved a 2.5x speedup on wire routing by optimizing on memory latencies.
    • Observed ~13x speedups as algorithm scales up to 64 threads. Higher thread counts see diminishing returns due to task distribution overheads.

    Parallel Wire Routing with MPI
    • Distributed the wire routing workload across multiple processes using MPI for cluster-based parallelism.
    • Implemented communication strategies to minimize synchronization overhead and maximize throughput.

    Parallel CUDA Renderer
    • Developed a GPU-accelerated renderer using CUDA to achieve real-time image synthesis.
    • Optimized memory access patterns and kernel launches for maximum performance.

    Lock-free Priority-aware Work Stealing
    • Implemented a lock-free work stealing scheduler supporting task priorities.
    • Demonstrated improved load balancing and reduced contention in parallel workloads.

    • Enhanced throughput to 12,800 kilo-OPS/sec by managing free blocks in segregated lists.

15-441/15-641: Networking and the Internet

    • Simulated a dynamic network with real-time routing updates and recomputed shortest paths after topology changes.
    • Implemented spanning tree protocol to prevent loop storms and improved privacy amongst nodes using random routing and buffer mechanism.

    HTTP Server/Client

    • Resolved resource dependencies using topological sort, enabling 60% faster page loads.
